Cybersecurity Note #5

Password Security & Multi-Factor Authentication (MFA)

Introduction

Passwords are one of the most common methods used to protect online accounts and sensitive information.

However, weak passwords can make it easy for attackers to gain unauthorized access.

Using strong passwords and enabling Multi-Factor Authentication (MFA) significantly improves security.


What Makes a Strong Password?

A strong password is difficult for attackers to guess or crack.

Characteristics of a strong password include:

* At least 12 characters long
* Combination of uppercase and lowercase letters
* Includes numbers
* Includes special characters
* Does not contain personal information

Example

Weak Password:

password123

Strong Password:

T7!mK#29vL@q


Common Password Mistakes

Many people unknowingly make their accounts vulnerable by:

* Reusing the same password across multiple accounts
* Using simple or predictable passwords
* Sharing passwords with others
* Writing passwords where others can easily find them



What is a Password Manager?

A password manager is a tool that securely stores and manages passwords.

Benefits include:

* Generates strong passwords
* Stores passwords securely
* Reduces the need to remember multiple passwords

Examples include:

* Bitwarden
* 1Password
* LastPass



What is Multi-Factor Authentication (MFA)?

Multi-Factor Authentication (MFA) adds an extra layer of security by requiring more than one form of verification.

Instead of relying only on a password, MFA requires an additional factor.

Examples include:

* One-Time Password (OTP)
* Authentication App
* Fingerprint Verification
* Facial Recognition



Why MFA is Important

Even if an attacker steals a password, MFA makes it much harder for them to access the account.

MFA helps protect:

* Email accounts
* Banking applications
* Social media accounts
* Work-related systems


Best Practices

To improve account security:

* Use strong and unique passwords
* Enable MFA whenever possible
* Avoid password reuse
* Use a password manager
* Update passwords when necessary


Key Takeaway

Passwords are the first line of defense for many online accounts.

Strong passwords combined with Multi-Factor Authentication provide significantly better protection against unauthorized access and cyber attacks.
